# Practica 1

Esta practica se enfoco en como leer imagenes, modificar sus atributos y obtener datos de estos

## Estructura y Definicion de los archivos

- Reto 1  

    - matricesImagenes.py → crea una matriz de 1000x1000 y saca datos relevantes como el min, max, media y desviacion estandar

    - main_simple.py → crea la matriz pero no genera la imagen, solo muestra los datos relevantes y tambien los tiempos de ejecucion para comparar con otros algoritmos

- Reto 2 → es una hoja de excel donde se pone en practica el uso de kernels, en este reto se encuentra una imagen creada con las celdas, cada una usada como un pixel y los kernels filtran o trasnforman dicha imagen

- Reto 5 

    - Convoluciones → cargamos y leemos una imagen, la trasnformamos a escala de grises, la pasamos por la funcion salt and pepper para añadir ruido, apartir de aca aplicamos distintas funciones para eliminar ese ruido y comparar cual de estas trabaja mejor
